OpenSource

KontrolVM is OpenSource and available on GitHub. If you want to add a feature fork it and get involved! If you want to request one submit a ticket!

Security Minded

KontrolVM aims to provide a beautiful solution to KVM management without putting any of your users or resources at risk. If you don't believe us, check out the source!

Free, as in Beer

Stop paying for products where you have no kontrol over their future!

Node based. Control more with less.

KontrolVM is designed to manage multiple, remote Hypervisors. This means that you can administrate your Hypervisors globally from a single web application. To ensure security we recommend that the Hypervisors are connected to KontrolVM via a VPN.

Highly configurable. Manage every aspect of each Hypervisor.

With KontrolVM you are able to set resource limits per Hypervisor, manage Storage Pools and Networks as well as individual Instances.

Users first. Designed to put User's experience first.

A normal User's User Interface has boiled down to it's bare bones in order to let you get your Instance up and running as quick as possible. For Admin Users it's designed to provide consistency across it's workflows.

Open. From OpenSource to OpenSource.

KontrolVM is built with the finest OpenSource ingredients. We use Django, Bootstrap, libvirt and KVM. With a sprinkling of magic third party libraries described in the repo's README.
